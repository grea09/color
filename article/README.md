# Dependencies

Packages to install on Ubuntu 14.04 (sudo apt-get install) :
* texlive-science
* texlive-fonts-recommended
* texlive-latex-extra
* xzdec
* pandoc
* pandoc-citeproc
* inotify-tools

CTAN package (tlmgr init-usertree && tlmgr install ):
* xcolor-solarized

Python packages (pip install) :
* pandoc-fignos
